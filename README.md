# 🎬 IMDB Sentiment Analysis — Machine Learning Pipeline

## 📌 Project Overview
This project builds an end-to-end Natural Language Processing (NLP) pipeline to classify movie reviews as **positive** or **negative** using machine learning.

The workflow demonstrates text preprocessing, feature extraction using Bag-of-Words, model training, evaluation, and creation of a reusable ML pipeline for sentiment prediction.

---

## 📊 Dataset
The project uses the IMDB movie review dataset, a widely used benchmark dataset for binary sentiment classification.

It contains labeled movie reviews that reflect real user opinions and language patterns.

---

---

## ⚙️ Project Workflow

### 1️⃣ Text Preprocessing
- Lowercasing text  
- Removing punctuation  
- Removing stopwords  
- Cleaning review content  

### 2️⃣ Feature Engineering
- Bag-of-Words using **CountVectorizer**  
- Converting text into numerical feature vectors  

### 3️⃣ Modeling
- Train-test split  
- Logistic Regression classifier  
- Integration into a reusable ML pipeline  

### 4️⃣ Evaluation
- Accuracy score  
- Confusion matrix  
- Classification report  

---

## 🔎 Key Insights
- Proper text preprocessing significantly improves model performance.  
- Bag-of-Words effectively captures sentiment patterns in reviews.  
- Logistic Regression provides a strong baseline for NLP classification tasks.  

---

## 📈 Results
The model successfully predicts sentiment polarity of movie reviews and demonstrates how traditional machine learning approaches can solve NLP classification problems.
